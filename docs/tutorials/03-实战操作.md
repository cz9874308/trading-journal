## 📁 3.1 投资组合管理

### 为什么需要多个组合？

不同的交易策略，表现特征不同：

```
┌─────────────────────────────────────────────────────────────────────┐
│                      为什么要分组合？                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ❌ 不分组合                                                         │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  所有交易混在一起                                            │   │
│  │  ├── 日内交易                                                │   │
│  │  ├── 波段交易                                                │   │
│  │  ├── 长线投资                                                │   │
│  │  └── 打新股                                                  │   │
│  │                                                              │   │
│  │  问题：无法分辨哪种策略赚钱，哪种亏钱                         │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  ✅ 分组合                                                          │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  组合 1: 日内交易     → 胜率 65%，平均持仓 2小时              │   │
│  │  组合 2: 波段交易     → 胜率 45%，平均持仓 3天                │   │
│  │  组合 3: 长线投资     → 胜率 80%，平均持仓 3个月              │   │
│  │                                                              │   │
│  │  结论：日内和长线表现好，波段需要改进                         │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 推荐的组合划分方式

| 划分维度 | 示例组合 | 适用场景 |
|----------|----------|----------|
| **按策略** | 突破策略、均值回归、趋势跟踪 | 测试不同策略效果 |
| **按时间框架** | 日内交易、波段交易、长线投资 | 区分不同持仓周期 |
| **按市场** | NSE 股票、BSE 股票、衍生品 | 区分不同市场表现 |
| **按账户** | 主账户、模拟账户、练习账户 | 区分真实和模拟 |

### 初始资金设置

初始资金用于计算收益率：

```
收益率 = 总盈亏 / 初始资金 × 100%

例如：
  初始资金：₹100,000
  总盈亏：₹5,000
  收益率：5%
```

> 💡 **建议**：设置真实的初始资金，便于计算真实收益率。

### 交易笔记模板

一个好的交易笔记应该回答以下问题：

```markdown
## 入场理由
- 为什么选择这只股票？
- 技术面信号是什么？
- 基本面有什么支撑？

## 交易计划
- 入场价位：
- 止损价位：
- 止盈目标：
- 风险回报比：

## 出场总结
- 实际出场价位：
- 是否按计划执行？
- 有什么可以改进的地方？

## 心理状态
- 入场时情绪如何？
- 持仓期间有无恐惧/贪婪？
- 出场决策是否理性？
```

**简化版模板**：

```
入场：[理由]
计划：止损 [价格]，止盈 [价格]
出场：[结果] - [反思]
```

## 📊 3.4 数据分析技巧

### 如何解读分析数据？

```
┌─────────────────────────────────────────────────────────────────────┐
│                     分析指标解读                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  🎯 胜率 (Win Rate)                                                  │
│  ├── > 50%  →  大多数交易盈利，但不一定整体盈利                     │
│  ├── < 50%  →  大多数交易亏损，但不一定整体亏损                     │
│  └── 关键：胜率要配合盈亏比看                                       │
│                                                                     │
│  💰 盈利因子 (Profit Factor)                                         │
│  ├── > 2.0  →  优秀，每亏 1 元赚 2 元以上                           │
│  ├── 1.5~2.0 →  良好，稳定盈利                                      │
│  ├── 1.0~1.5 →  一般，勉强盈利                                      │
│  └── < 1.0  →  亏损，需要改进                                       │
│                                                                     │
│  📈 平均盈亏比                                                       │
│  ├── 平均盈利 / |平均亏损|                                          │
│  ├── > 2.0  →  每笔盈利是亏损的 2 倍以上                            │
│  └── 配合胜率：低胜率高盈亏比 或 高胜率低盈亏比 都可以盈利          │
│                                                                     │
│  📊 实际例子                                                         │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  交易员 A                    交易员 B                        │   │
│  │  胜率：30%                   胜率：70%                       │   │
│  │  平均盈利：₹3,000            平均盈利：₹500                  │   │
│  │  平均亏损：₹500              平均亏损：₹1,000                │   │
│  │                                                              │   │
│  │  10 笔交易：                 10 笔交易：                      │   │
│  │  3 笔盈利：₹9,000            7 笔盈利：₹3,500                │   │
│  │  7 笔亏损：₹3,500            3 笔亏损：₹3,000                │   │
│  │  净盈利：₹5,500 ✅           净盈利：₹500 ✅                 │   │
│  │                                                              │   │
│  │  结论：低胜率也能盈利，关键是盈亏比！                        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 按股票分析

Analytics 页面的 "By Symbol" 功能可以帮你发现：

| 发现 | 行动 |
|------|------|
| 某股票胜率特别高 | 可以考虑增加该股票的交易 |
| 某股票胜率特别低 | 分析原因，考虑避开或改进 |
| 某股票交易次数很多但亏损 | 可能存在过度交易 |

### 常见错误及改进

| 错误 | 后果 | 改进方法 |
|------|------|----------|
| 不及时记录 | 忘记细节，数据不准 | 入场后立即记录 |
| 不写笔记 | 无法复盘改进 | 至少写入场理由 |
| 不截图 | 无法回顾图形 | 形成截图习惯 |
| 不用标签 | 难以分类分析 | 建立标签体系 |
| 不看分析 | 浪费数据价值 | 每周查看一次 |

## 📌 核心要点总结

1. **分组合管理不同策略，便于分析和改进**
2. **完整记录 = 基本信息 + 标签 + 笔记 + 截图**
3. **标签体系帮助你分类和筛选交易**
4. **胜率和盈亏比要配合看，高盈亏比可以弥补低胜率**
5. **坚持记录是关键，从简单开始，逐步完善**

## 📖 课程目录

### [第 1 课：快速开始](./01-快速开始.md)

**⏱ 30 分钟 | ⭐ 零基础入门**

从零开始，成功运行项目并创建第一笔交易记录。

学习内容：
- 环境准备和安装
- 使用 Docker 一键启动
- 注册账户和登录
- 创建投资组合
- 记录第一笔交易
- 平仓并查看盈亏

---

### [第 2 课：核心概念理解](./02-核心概念.md)

**⏱ 45 分钟 | ⭐⭐ 基础进阶**

理解项目架构、前后端通信和数据流。

学习内容：
- 全栈应用的组成
- 项目目录结构
- API 接口说明
- 数据库设计
- 认证与安全机制
- 盈亏计算逻辑

---

### [第 3 课：实战操作指南](./03-实战操作.md)

**⏱ 60 分钟 | ⭐⭐⭐ 实战应用**

掌握投资组合管理、交易记录和数据分析的最佳实践。

学习内容：
- 投资组合规划
- 交易记录规范
- 标签体系设计
- 截图使用技巧
- 数据分析解读
- 交易习惯养成

---

### [第 4 课：常见问题解答](./04-常见问题.md)

**⏱ 按需查阅 | ⭐ 参考手册**

遇到问题时的故障排除指南。

包含内容：
- 启动问题排查
- 登录问题解决
- 功能使用疑问
- 数据备份恢复
- 进阶配置说明

## 🚀 快速开始

如果你时间有限，可以按以下顺序学习：

1. **30 分钟快速入门**
   - 阅读 [第 1 课](./01-快速开始.md)
   - 完成项目启动和第一笔交易

2. **深入理解**
   - 阅读 [第 2 课](./02-核心概念.md)
   - 理解系统如何工作

3. **熟练使用**
   - 阅读 [第 3 课](./03-实战操作.md)
   - 建立自己的使用习惯

4. **遇到问题**
   - 查阅 [第 4 课](./04-常见问题.md)
   - 找到解决方案

## 📊 技术栈概览

如果你对技术感兴趣，这是项目使用的技术栈：

| 层级 | 技术 | 用途 |
|------|------|------|
| **前端** | Next.js 14 | React 框架 |
| | TypeScript | 类型安全 |
| | Tailwind CSS | 样式框架 |
| | shadcn/ui | UI 组件库 |
| | Zustand | 状态管理 |
| **后端** | FastAPI | Python Web 框架 |
| | SQLAlchemy | ORM 框架 |
| | Pydantic | 数据验证 |
| | JWT | 身份认证 |
| **数据库** | SQLite | 轻量级数据库 |
| **部署** | Docker | 容器化部署 |

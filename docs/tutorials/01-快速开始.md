## 🤔 这是什么项目？

**Vibe Journal** 是一个帮助交易者记录和分析交易的工具。

想象一下：

> 你每天做交易，买入卖出各种股票。一个月后，你想知道：
> - 我这个月赚了多少钱？
> - 哪些股票让我赚钱了？哪些亏钱了？
> - 我的胜率是多少？

如果没有记录，这些问题很难回答。

**Vibe Journal 就是帮你解决这个问题的！**

```
┌─────────────────────────────────────────────────────────┐
│                    Vibe Journal 能做什么？                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  📝 记录交易                                             │
│     → 入场价格、数量、时间                                │
│     → 出场价格、盈亏计算                                  │
│     → 交易笔记和截图                                     │
│                                                         │
│  📊 分析表现                                             │
│     → 总盈亏统计                                         │
│     → 胜率计算                                           │
│     → 最佳/最差交易                                      │
│                                                         │
│  📁 管理组合                                             │
│     → 创建多个投资组合                                    │
│     → 分类管理不同策略                                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 🛠 Step 1：准备工作

在开始之前，请确保你的电脑已安装以下软件：

### 方式一：使用 Docker（推荐新手）

只需要安装一个软件：

| 软件 | 用途 | 下载地址 |
|------|------|----------|
| **Docker Desktop** | 一键运行整个项目 | [点击下载](https://www.docker.com/products/docker-desktop/) |

> 💡 **小贴士**：Docker 就像一个"虚拟电脑"，帮你自动配置好所有环境，省去很多麻烦。

### 方式二：手动安装（适合想深入学习的同学）

需要安装两个软件：

| 软件 | 版本要求 | 用途 | 下载地址 |
|------|----------|------|----------|
| **Python** | 3.10 以上 | 运行后端服务 | [点击下载](https://www.python.org/downloads/) |
| **Node.js** | 18 以上 | 运行前端页面 | [点击下载](https://nodejs.org/) |

## 🚀 Step 2：下载项目代码

打开命令行（Windows 按 `Win + R`，输入 `cmd` 回车），输入：

```bash
# 进入你想存放项目的目录
cd D:\projects

# 下载代码（如果你有 git）
git clone https://github.com/你的用户名/trading-journal.git

# 进入项目目录
cd trading-journal
```

> 📌 如果没有 git，也可以直接下载 ZIP 文件并解压。

## 💻 Step 3-B：手动启动（可选）

如果你想手动启动，需要打开**两个命令行窗口**。

### 窗口 1：启动后端

```bash
# 进入后端目录
cd backend

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 启动后端
python run.py
```

看到以下输出说明后端启动成功：

```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete.
```

### 窗口 2：启动前端

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动前端
npm run dev
```

看到以下输出说明前端启动成功：

```
▲ Next.js 14.x.x
- Local: http://localhost:3000
✓ Ready in 2s
```

## 👤 Step 5：注册账户

点击 **"点击注册"** 进入注册页面：

1. **填写注册信息**：

   | 字段 | 示例 | 说明 |
   |------|------|------|
   | Email | trader@example.com | 用于登录 |
   | Username | trader001 | 用户名 |
   | Password | MySecure123! | 至少 6 位 |
   | Full Name | 张三（可选） | 显示名称 |

2. **点击"Create Account"注册**

> ⭐ **重要提示**：第一个注册的用户会自动成为**管理员**！

## 📂 Step 6：创建投资组合

登录后，你会看到空的仪表板。让我们创建第一个投资组合！

1. **点击 "New Portfolio" 按钮**

2. **填写组合信息**：

   ```
   ┌─────────────────────────────────────────────────────┐
   │              创建新投资组合                          │
   ├─────────────────────────────────────────────────────┤
   │                                                     │
   │   名称：    [ 我的日内交易___________ ]             │
   │                                                     │
   │   描述：    [ 用于练习日内交易策略_____ ]           │
   │                                                     │
   │   初始资金： [ 100000_________________ ] ₹          │
   │                                                     │
   │                  [ 创建组合 ]                        │
   │                                                     │
   └─────────────────────────────────────────────────────┘
   ```

3. **点击"创建组合"**

> 💡 **建议**：可以创建多个组合来管理不同的交易策略，比如"日内交易"、"波段交易"、"长线投资"等。

## ✅ Step 8：平仓并查看盈亏

当你想结束这笔交易时：

1. **点击交易记录**

2. **点击 "Close Trade" 按钮**

3. **填写出场信息**：

   | 字段 | 示例 |
   |------|------|
   | Exit Price | 2600 |
   | Exit Date | 2024-01-15 15:00 |

4. **点击确认**

系统会自动计算盈亏：

回答以下问题，检验你的学习成果：

1. **项目由哪两个部分组成？分别运行在哪个端口？**
   
2. **第一个注册的用户有什么特殊身份？**

3. **交易的盈亏是如何计算的？**

4. **做多(Long)和做空(Short)有什么区别？**

<details>
<summary>点击查看答案</summary>

1. 后端(FastAPI, 8000端口) + 前端(Next.js, 3000端口)
2. 自动成为管理员(Admin)
3. 做多盈亏 = (出场价 - 入场价) × 数量
4. 做多是买入等涨再卖；做空是借入卖出等跌再买回

</details>

## 🔗 常见问题

### Q: 启动时报错怎么办？

A: 查看 [04-常见问题.md](./04-常见问题.md) 的故障排除章节。

### Q: 如何重置数据库？

A: 删除 `backend/trade_journal.db` 文件，重启后端即可。

### Q: 忘记密码怎么办？

A: 目前版本暂不支持密码重置，需要管理员协助或重置数据库。
